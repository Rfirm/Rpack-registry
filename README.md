#Rpack-registry
Manage account.

##Document
###start the django
First you have to install the django and django-rest-framework

Create database

    python manage.py syncdb

Start server

    python manage.py runserver

###install package
1. django bower

Install `bower` from `npm`

    npm install -g bower

Then install `django-bower` package

    pip install django-bower
