# Rpack-registry

**Working in process**

Require:

- Django: 1.6.3
- Python: 2.x

Registry for Rpack




### Create a project using django

```
python /usr/local/lib/python2.7/site-packages/django/bin/django-admin.py startproject rpack_register
```

### Start server

```
python manage.py runserver
```

### Sync database

```
python manage.py syncdb
```

### Setting database

Using sqlite3
